# Unlocked-BOOMS-

Unlocked Booms is a custom mod for the Furby Boom that unlocks hidden or normally restricted “booms” (features, behaviors, and sounds). The mod allows users to activate multiple booms that the original system usually keeps locked or limited.

With Unlocked Booms installed, the Furby can access extra interactions, experimental behaviors, and expanded responses. Because it removes some of the normal limits, different booms may run at the same time, which can sometimes cause unusual or unpredictable behavior.

Features
- Unlocks hidden boom features
- Enables additional sounds and responses
- Allows experimental interactions
- Removes certain built-in feature restrictions

WARNING
Installing Unlocked Booms will overwrite the RAM data of your Furby Boom.

This means:
- The Furby’s stored behavior data may change
- Some original settings may be lost
- The Furby may behave differently than normal

The only way to return the Furby to its original state is to perform a full RAM reset.

A full reset will completely clear all stored data, including:
- The Furby’s name
- Learned behaviors
- Stored interactions
- Custom settings

After the reset, the Furby Boom will start fresh as if it were new.

Summary
Unlocked Booms expands the capabilities of Furby Boom by unlocking hidden features and allowing experimental behaviors, but it may permanently modify stored data unless the device is fully reset.
